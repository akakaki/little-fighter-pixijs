# Little Fighter pixiJS

![](/public/logo.png)

用 pixi 復刻經典遊戲 Little Fighter 2。

## 按鍵操作
- [ ] 規劃 keyboard 的按下、長按與放開
- [ ] 設定密碼秒數與其他輸入保留長度不相同
 
## 角色
- [ ] 使用者輸入按鍵欄位
- [ ] 角色、物品動畫
- [ ] 按鈕消耗後返還至按鍵紀錄內
- [ ] 角色與技能腳本轉譯成 json
- [ ] 碰撞與數值消耗
- [ ] 電腦 ai 設定靠近敵人 依照技能範圍選擇釋放
- [ ] 使用者指揮同隊電腦

## 環境
- [ ] 背景可滾動與視覺差
- [ ] 製作連線對戰
